# 就诊数据采集系统 - 测试版本

## 项目说明
这是一个使用Python + Kivy开发的跨平台移动应用原型，用于测试和预览就诊数据采集系统的界面和功能。

## 功能模块

### 1. 首页 (Home)
- 系统标题和简介
- 四个功能入口按钮
- 状态显示

### 2. NFC身份验证 (NFC)
- 身份证扫描界面
- 扫描状态显示
- 身份信息展示

### 3. 摄像头识别 (Camera)
- 摄像头预览区域（模拟）
- 拍照识别按钮
- 人脸识别结果显示

### 4. 症状语音记录 (Voice)
- 录音控制按钮
- 录音状态显示
- 语音转文字结果

### 5. 数据汇总 (Summary)
- 采集数据概览
- 患者信息展示
- 数据上传功能

## 运行方法

### 1. 安装依赖
```bash
pip install -r requirements.txt
```

### 2. 运行应用
```bash
python main.py
```

### 3. 查看效果
- 窗口大小：360x640（模拟手机屏幕）
- 点击各个按钮测试功能
- 查看页面切换效果

## 界面预览
